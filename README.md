## Feature
```
- Retrieve Books from GQL

```

## Getting Started
mac
```
$ git clone https://github.com/JNU-SoftwareLAB/GQL_practice.git
$ cd GQL_practice
$ npm run mac
```

window
```
$ git clone https://github.com/JNU-SoftwareLAB/GQL_practice.git
$ cd GQL_practice
$ npm run window
```
